Programming 02

Programming 02 builds on your Java programming from Programming 01 to develop your knowledge of object oriented programming.

Content

Object Oriented Programming (OOP)

Object Oriented Programming is a programming approach that considers systems to be made up of objects that interact with each other.

An object is made up of data (information about the object) and methods (things the object can do).

Classes & Objects

We will look at how to design classes, the templates used to build objects. Multiple objects can be built from a single class.

OOP Properties

We will examine the four properties that power object oriented programming - abstraction, encapsulation, polymorphism and inheritance.

OOA and OOD

We will examine how to analyse problems to design object oriented software
