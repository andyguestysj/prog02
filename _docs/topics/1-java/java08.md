### Operations

|Operator|Operation|Example|
|---|---|---|
|+|addition|`int x = 5 + 4;`|
|-|subtraction|`int x = 5 - 4;`|
|*|multiplication|`int x = 5 * 4;`|
|/|division|`float x = 5.3 / 4.1;`|
|/|division|`int x = 5 / 4;` (Integer division!)|
|%|modulus|`int x = 5 % 4;` (Integer division remainder!)|
|++|increment|`x++;`|
|--|decrement|`x--;`|

### Comparisons

|Operator|Comparison|
|---|---|
|==| equality |
|!=| inequality |
|>|greater than|
|>=|greater than or equal|
|<|less than|
|<=|less than or equal|

### Logic

|Operator|Operation|
|---|---|
|&&|logical AND|
|\|\||logical OR|
